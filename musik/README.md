# Musik Ordner

Hier sollten deine Musikdateien gespeichert werden.

Unterstützte Formate:
- MP3 (.mp3)
- WAV (.wav)  
- OGG (.ogg)
- M4A (.m4a)

Einfach Musikdateien in diesen Ordner kopieren und sie werden im Musikplayer verfügbar sein.

## Playlists

Playlists werden automatisch im 'playlist' Ordner gespeichert als JSON-Dateien.
