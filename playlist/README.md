# Playlists Ordner

Hier werden automatisch deine Playlists gespeichert.

Jede Playlist ist eine JSON-Datei mit folgendem Format:
```json
{
  "name": "Meine Playlist",
  "songs": [
    "pfad/zur/musik1.mp3",
    "pfad/zur/musik2.wav"
  ]
}
```
